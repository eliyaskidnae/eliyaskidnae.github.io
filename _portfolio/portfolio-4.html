GitHub Report Video " collection: portfolio --- This project integrates **planning, perception, localization**, and **control** to enable fully autonomous exploration and object manipulation in unknown environments. The robot dynamically builds a map using a high-resolution 2D LiDAR while localizing itself in real time. A **Dubins-path-based RRT\*** planner generates collision-free trajectories, allowing the robot to safely navigate around obstacles while continuously updating its map. As part of the exploration, the robot also performs object detection. When an object is identified, it autonomously approaches, picks it up, and places it at a predefined drop-off location. After completing the placement, it resumes its exploration and mapping task. This seamless integration of motion planning, environment perception, and manipulation demonstrates a robust and adaptable robotic system capable of performing complex tasks in unstructured environments.
