GitHub Report Video " collection: portfolio --- Autonomous exploration and navigation in unknown environments present significant challenges for robotics—especially for differential drive robots, which require dynamically feasible and efficient path planning in real time. This project presents a robust software architecture designed specifically for such scenarios, integrating key technologies to enhance robotic autonomy. At the heart of the system is a frontier-based exploration strategy that uses RRT* (Rapidly-exploring Random Trees Star) tailored with a Dubins state space, ensuring the generation of smooth, feasible, and collision-free trajectories suitable for non-holonomic robots. The robot’s perception system relies on a high-resolution 2D LiDAR, enabling accurate obstacle detection and real-time mapping of the environment. The global path planner employs RRT* enhanced with a custom cost function and post-processing optimization to improve path quality. To decide on the next frontier to explore, a greedy selection algorithm prioritizes locally valuable regions, effectively minimizing traversal cost and improving exploration efficiency. Together, these components create a system capable of efficiently covering unknown environments with a strong balance between global planning and local adaptability.
