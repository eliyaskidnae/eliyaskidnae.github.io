GitHub Report Video " collection: portfolio --- Autonomous navigation in unknown environments is a major challenge in robotics, especially for differential drive robots that must follow realistic motion constraints while avoiding obstacles and exploring efficiently. This project develops an autonomous exploration system using a 2D LiDAR sensor for real-time mapping and obstacle detection. The robot uses frontier-based exploration to identify unexplored areas and RRT* with a Dubins state space to generate smooth and feasible paths. A greedy frontier selection strategy is used to improve exploration efficiency and reduce unnecessary movement. The system combines mapping, exploration, and motion-aware path planning to enable efficient autonomous navigation in unknown environments.
