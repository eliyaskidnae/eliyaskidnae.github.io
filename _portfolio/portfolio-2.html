--- title: "Pose Based Graph SLAM Using Scan Matching with ICP(Mobile Robot Localization )" excerpt: "This project research focuses on the implementation of an Extended Kalman Filter (EKF) SLAM algorithm using the Iterative Closest Point (ICP) method for scan matching.The algorithm is implemented on a Kobuki Turtlebot, a widely-used mobile robot platform known for its robustness and versatility. The Turtlebot is equipped with a 2D LIDAR, IMU, and wheel encoders, which provide the necessary sensor data for SLAM.
