--- title: "Pose Based Graph SLAM Using Scan Matching with ICP(Mobile Robot Localization)" excerpt: " Reliable localization and mapping are fundamental challenges in mobile robotics, especially in unknown environments where robots must estimate their position while simultaneously building a map of the surroundings. This project focuses on implementing an EKF-SLAM system on a Kobuki Turtlebot equipped with a 2D LiDAR, IMU, and wheel encoders. The IMU and wheel encoder data are used for motion estimation, while LiDAR data is used for environmental perception and scan matching. To improve localization accuracy, the system applies the Iterative Closest Point (ICP) algorithm to align consecutive LiDAR scans. By combining sensor fusion with scan matching, the robot can estimate its pose more accurately while building a consistent map of the environment in real time.
